1. Home Page: The Hub

Designed as the central navigation hub, focusing on visual hierarchy and immediate information access.

  • Hero Section: Introduces professional identity immediately.
  • CSS Grid Layout: The "Research Interests" section utilizes a responsive grid that adapts to screen width.
  • Skip Link: Implemented "Skip to Main Content" for keyboard accessibility.
  • Navigation: Consistent header linking to all sub-pages.

2. Research Page: Visual Gallery

Showcases academic projects using a card-based layout tailored for visual density.

  • Card Layout: Each project is encapsulated in a semantic article tag.
  • Image Heavy: Optimized for displaying maps and diagrams related to Geospatial Data Science.
  • Interactivity: Features click-to-expand functionality (detailed in Slide 7).
  • Semantic HTML: Used <figure> and <figcaption> for proper image semantics.
3 / 8

3. Publications: Structured List

A clean, text-forward layout designed for readability and quick scanning of citations.

  • Typography: prioritized legibility with high contrast and proper line-height.
  • External Links: Clear visual indicators for DOI and PDF links.
  • Reverse Chronological: Logic ordering of content for academic relevance.
  • Responsive: Text wraps and stacks gracefully on mobile devices.
4 / 8

4. Experience: Timeline Layout

Displays professional history using a structured, semantic approach.

  • Flexbox Architecture: Uses CSS Flexbox to align dates (left) and details (right).
  • Mobile Adaptation: On screens < 768px, the layout shifts to a vertical stack for better readability.
  • Hierarchy: Clear distinction between Role, Company, and Description using h3 and h4 tags.
5 / 8

5. Contact: Forms & Layout

A functional gateway for professional connection.

  • Accessible Forms: All inputs have associated <label> tags (Explicit association).
  • Focus States: Custom CSS focus rings for keyboard navigation visibility.
  • Layout: Centralized container with max-width for readability on large screens.
  • Social Links: Icon-based links to LinkedIn and GitHub with aria-label attributes.

Accessibility Compliance (POUR)

The site was rigorously tested against WCAG 2.1 AA standards.

Perceivable

- All non-text content (images) has alt text.
- Color contrast ratios meet or exceed 4.5:1 (UMich Blue/Maize/White).

Operable

- "Skip to Main Content" link provided.
- All interactive elements (Modals, Forms) are keyboard navigable.

Understandable

- Consistent navigation across all 5 pages.
- Error prevention in forms (Required fields marked).

Robust

- Valid HTML5 markup (checked via W3C Validator).
- Compatible with screen readers and mobile browsers.
